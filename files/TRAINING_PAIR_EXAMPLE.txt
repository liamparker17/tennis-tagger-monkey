╔══════════════════════════════════════════════════════════════╗
║   🔄 WHAT HAPPENS BEHIND THE SCENES                          ║
╚══════════════════════════════════════════════════════════════╝

When you click "Submit Video+CSV Pair":

1. System checks: Do video and CSV have matching names?
   finals_match.mp4 ←→ finals_match.csv ✓

2. Creates timestamped folder:
   data/training_pairs/finals_match_20250111_143052/

3. Copies files:
   → finals_match.mp4 copied to folder
   → finals_match.csv copied to folder

4. Reads CSV and counts tags:
   → Found 182 rows in CSV
   → Each row = 1 tag/event

5. Shows success message with info


╔══════════════════════════════════════════════════════════════╗
║   ❌ ERROR MESSAGES YOU MIGHT SEE                            ║
╚══════════════════════════════════════════════════════════════╝

Error 1: Missing Files
----------------------
❌ Error: Please upload both video and CSV files

→ Solution: Upload BOTH files before clicking submit


Error 2: Filename Mismatch
--------------------------
❌ Error: No matching video-CSV pairs found.

Videos: match1, match2
CSVs: game1, game2

Files must have matching names (e.g., match1.mp4 + match1.csv)

→ Solution: Rename files so they match:
  match1.mp4 → match1.csv
  match2.mp4 → match2.csv


Error 3: File Save Error
-------------------------
❌ Error saving pair 'finals_match': [Permission denied]

→ Solution: Check file permissions, make sure data/ folder is writable

TIP 1: Upload Multiple Pairs
-----------------------------
You can upload multiple videos and CSVs at once!

Videos:
  - match1.mp4
  - match2.mp4
  - match3.mp4

CSVs:
  - match1.csv
  - match2.csv
  - match3.csv

Click submit → All 3 pairs saved!


TIP 2: Consistent Naming
------------------------
Use a naming convention:

  date_opponent_match.mp4
  date_opponent_match.csv

Example:
  20250111_smith_singles.mp4
  20250111_smith_singles.csv


TIP 3: Quality Over Quantity
-----------------------------
Better to have:
  ✓ 10 perfectly tagged matches
Than:
  ✗ 100 poorly tagged matches

The model learns from your examples!


TIP 4: Build a Library
----------------------
Keep submitting good examples over time:

Week 1: 5 matches
Week 2: 8 matches
Week 3: 10 matches

= 23 high-quality training examples!

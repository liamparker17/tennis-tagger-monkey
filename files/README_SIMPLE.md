# 🎾 Tennis Tagger - Simple Start Guide

## 🚀 How to Start

### **Start Everything (Recommended)**
Double-click: **`START_HERE.bat`**

Opens 2 desktop windows:
1. Training System v3.1
2. Video Tagging System

---

### **Start Only Training**
Double-click: **`start_training_desktop.bat`**

---

### **Start Only Tagging**
Double-click: **`start_tagging_desktop.bat`**

---

### **Build Standalone EXE Files**
Double-click: **`build_exe.bat`**

Creates:
- `TennisTagger_Training.exe`
- `TennisTagger_Tagging.exe`

---

## 📁 Files You Need to Know

| File | What It Does |
|------|-------------|
| **`START_HERE.bat`** | ⭐ Use this to start both apps |
| `start_training_desktop.bat` | Training only |
| `start_tagging_desktop.bat` | Tagging only |
| `build_exe.bat` | Build standalone EXE files |

**All other .bat files have been removed!**

---

## ⚠️ First Time Use

When you run START_HERE.bat for the first time:

1. **Two terminal windows will pop up** (don't close them!)
2. Each shows server startup messages
3. Wait 10-15 seconds
4. **Desktop windows will appear** with the actual interfaces
5. Done!

---

## 🐛 If Nothing Happens

1. Check the terminal windows for error messages
2. Make sure Python is installed: `python --version`
3. Make sure you have dependencies: The bat file auto-installs them
4. Wait longer - first startup can take 15-20 seconds

---

## 💡 What's Different

**Each app is now STANDALONE:**
- Training runs independently on port 7861
- Tagging runs independently on port 7860
- Close one, the other keeps running
- No launcher screen needed

**Desktop Mode:**
- Opens in native Windows windows (not browser)
- Looks like a real desktop application
- Can be packaged as EXE files

---

## ✅ That's It!

Just double-click **`START_HERE.bat`** and you're good to go!

Wait for the desktop windows to appear (10-15 seconds).

🚀 HOW TO START:

Simply double-click:

   START_HERE.bat

This unified launcher gives you options to:
   [1] Launch Video Tagging System
   [2] Launch Training System
   [3] Diagnose PyWebView Issues
   [4] Run Step-by-Step Tests
   [5] Exit


📋 BEFORE FIRST RUN:
--------------------
1. Make sure Python 3.11+ is installed
2. Activate your virtual environment (done automatically by launcher)
3. If PyWebView doesn't display a window:
   - Choose option [3] from the launcher to diagnose
   - You may need to install WebView2 Runtime:
     https://go.microsoft.com/fwlink/p/?LinkId=2124703


🔧 TROUBLESHOOTING:
-------------------
If you get errors:

1. "Gradio compatibility error":
   ✓ FIXED! The 'info' parameters have been removed

2. "Window doesn't open":
   - Run START_HERE.bat → Choose [3] to diagnose
   - Install WebView2 Runtime if missing

3. "Server failed to start":
   - Check the terminal output for errors
   - Make sure ports 7860 and 7861 are not in use
   - Debug mode is now enabled, so you'll see detailed errors
